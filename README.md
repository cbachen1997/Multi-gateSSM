Basic code of SSM method based on multi-gate

Main dependent library version
====
###     tensorflow==2.1.0
###     keras==2.3.1
###     h5py==2.10.0

文件说明：
Read.me
====
###     1_ROI2Patches:Generating dataset of patches based on sample points from the roi image.
